Privacy Policy — Family Hub

Last updated: April 16, 2026  |  Contact: relwik@gmail.com

What Data We Collect

Calendar Data

  • Google Calendar events (synced via Google Calendar API)
  • Apple Calendar events (synced via Apple Sign-In + iCloud)
  • iCal/webcal feed events (fetched from URLs you provide)
  • Manually created events within the app

Task Data

  • Google Tasks lists and items (synced via Google Tasks API)
  • Locally created to-do lists and items

Location Data

  • Device GPS coordinates (used only for weather display)
  • City/region name (derived from GPS for weather location label)

Cleaning Data

  • Cleaning log entries (what was cleaned, by whom, when, and notes)
  • Stored entirely on-device

Member Data

  • Family member names, initials, colors, and roles
  • Optional 4-digit hub PIN (hashed with SHA-256 + random salt, stored in device secure storage — never stored or transmitted in plaintext)

Where Data Is Stored

All data is stored on-device only. Family Hub does not operate any cloud servers or databases.

  • Calendar/task/list data: AsyncStorage (on-device local storage)
  • Google/Apple OAuth tokens: Expo SecureStore (hardware-encrypted via Android Keystore / iOS Keychain)
  • Hub PIN hash: Expo SecureStore (hardware-encrypted)
  • Weather location: AsyncStorage (latitude, longitude, city name)
  • Cleaning logs: AsyncStorage (on-device)

Third-Party Services

Google Calendar API & Google Tasks API

  • Used to read and write calendar events and tasks
  • Data transmitted directly between your device and Google's servers
  • Subject to Google's Privacy Policy

Apple Sign-In & iCloud Calendar

  • Used to access iCloud calendars when you connect an Apple account
  • Subject to Apple's Privacy Policy

Open-Meteo Weather API

  • Used to fetch current weather conditions
  • Your device's GPS coordinates are sent to Open-Meteo to retrieve local weather
  • Open-Meteo is a free, open-source API — privacy info

Sentry (Crash Reporting)

  • Used to collect anonymous crash diagnostic data (device type, OS version, app version, stack trace)
  • No personally identifiable information is included in crash reports
  • Subject to Sentry's Privacy Policy

Data We Do NOT Collect

  • No behavioral analytics or usage tracking
  • No advertising identifiers
  • No data sold to third parties
  • No cloud storage or server-side databases
  • No biometric data

Security

  • Hub PINs are hashed with SHA-256 + random salt before storage
  • Brute-force protection: 5 wrong PIN attempts triggers exponential lockout
  • OAuth tokens stored in hardware-encrypted secure storage
  • All network traffic uses HTTPS/TLS (cleartext traffic is blocked)
  • Android backup disabled to prevent unauthorized data extraction
  • ProGuard code obfuscation enabled in release builds

How to Delete Your Data

  • Remove a family member: Settings > General > tap member > Delete
  • Clear all data: Settings > Security > Danger Zone > Reset All Data
  • Revoke Google access: Visit Google Account Permissions and remove FamilyHub
  • Uninstall the app: All local data is permanently deleted

Children's Privacy

Family Hub supports child member profiles managed by adult administrators. The app does not independently collect data from children under 13. All data is managed locally by the family's adult members.

Changes to This Policy

We may update this privacy policy from time to time. The "Last updated" date at the top indicates the most recent revision.
